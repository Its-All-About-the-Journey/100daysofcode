
# DAY 23

Sort of Frogger

# Description

Play not Frogger and cross the expanse with your turtle.

Make it to the other side if you can.

Watch out, the challenge increases!

# Environment
OS: Ubuntu Bionic

Python version: 3.8.7

# Dependencies

Core

# How to run script
```
Call the script.

Press "w" to move up
Press "s" to move back
```
