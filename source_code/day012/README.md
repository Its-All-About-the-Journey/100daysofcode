# DAY 12

Number Guessing Game

# Description

# Environment
OS: Ubuntu Bionic

Python version: 3.6.9

# Dependencies

None

# How to run script
```
Call the script. 

The computer will choose a number between 1 and 100.

Choose "Easy" or "Hard" to determine how many guesses you get.

Try and guess the number, before you run out of chances.

```
